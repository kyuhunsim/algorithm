1. try and except

<br/>

```python 
try:
    ### 어쩌고 저쩌고 ###
except:
    ### 어쩌고 저쩌고 ###
```
예외 처리 코드로 try에서 예외가 생겨서 error 가 나오면 except로 이동해서 처리
